===========
Largest Sum
===========

Find the contiguous subarray (having at least one number) with the largest sum in an integer array nums and return its sum. 
A subarray is a section of an array that is contiguous.

Sample
******
input.txt

.. code:: text

      nums = -2 1 -3 4 -1 2 1 -5 4
 
output.txt

.. code:: text

     6
